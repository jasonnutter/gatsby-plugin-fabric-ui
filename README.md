# gatsby-plugin-fabric-ui

Gatsby plugin for the Fabric UI framework.

## Installation

```sh
npm install gatsby-plugin-fabric-ui
```

### Usage

`gatsby-config.js`
```js
module.exports = {
    plugins: [
        'gatsby-plugin-fabric-ui'
    ]
}
```

## Compatibility

Currently only tested with:

* `react` v16
* `gatsby` v2
* `office-ui-fabric-react` v6
